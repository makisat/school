* Assignment
Convert each byte in inputBuf into two ASCII hex digits, add a trailing space,
replace the very last space with a newline, and print the entire line to
standard output.

Expected runtime output:

#+begin_example
83 6A 88 DE 9A C3 54 9A
#+end_example
